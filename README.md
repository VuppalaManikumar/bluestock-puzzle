# Bluestock Daily Puzzle – Brand Guidelines

This project implements Bluestock official brand colors.

## Brand Colors Used

Primary: #414BEA  
Secondary: #7752FE  
Accent: #F05537  
Dark Text: #222222  
Background: #F6F5F5  
White: #FFFFFF

## Implementation

The colors are defined using CSS variables and applied to:

- Navigation bar
- Buttons
- Page background
- UI components

## Files

index.html – main page  
brand.css – brand color styles
